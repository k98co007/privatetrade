# 유저스토리 문서 (UserStory)
**버전**: 2.0  
**기반**: TICKET-006 (4차 인터뷰)  
**작성일**: 2026년 2월 8일  
**변경 사항**: US-007 추가 (특정 종목 선택 기능), 기존 US-001~006 리뷰 완료  

---

## US-001: 종목 구성 편집 (블랙리스트)

**As a** 주식투자자  
**I want to** 코스피 200 종목 중 내가 원하지 않는 종목을 제외할 수 있도록  
**So that** 백테스팅이 관심 있는 종목에만 수행되어 시간을 절약할 수 있다

**수락 조건**:
- [x] 웹브라우저 UI에서 "블랙리스트 편집" 버튼 클릭 시 팝업 또는 실드 화면 표시
- [x] 현재 블랙리스트 목록 조회 가능
- [x] 종목 추가/제거 가능 (검색 또는 체크박스)
- [x] 저장 시 서버에 즉시 반영
- [x] 서버 재시작 후에도 블랙리스트 유지

---

## US-002: 종목 구성 편집 (화이트리스트)

**As a** 주식투자자  
**I want to** 코스피 200 외 추가 종목을 백테스팅 대상에 포함할 수 있도록  
**So that** 관심 있는 특정 종목들도 함께 백테스팅할 수 있다

**수락 조건**:
- [x] 웹브라우저 UI에서 "화이트리스트 추가" 버튼 클릭 시 입력 화면 표시
- [x] 종목코드 또는 종목명으로 검색 가능
- [x] 화이트리스트에 추가된 종목 목록 조회 가능
- [x] 종목 제거 가능
- [x] 저장 시 서버에 즉시 반영
- [x] 서버 재시작 후에도 화이트리스트 유지

---

## US-003: 투자 전략 선택

**As a** 주식투자자  
**I want to** 백테스팅 수행 전에 어떤 투자 전략을 테스트할지 선택할 수 있도록  
**So that** 다양한 투자 전략을 비교분석할 수 있다

**지원 전략**:
1. **전략 1**: 고정시간 매수/매도 (Daily Trading)
2. **전략 2**: Trailing Stop 전략 (Next-day Profit Target)

**수락 조건**:
- [x] 웹브라우저에서 라디오 버튼 또는 드롭다운으로 전략 선택 가능
- [x] 선택된 전략에 따라 추가 파라미터 입력 필드 표시
- [x] 전략마다 기본값 제시 (예: 기본 매수시간, 매도시간)
- [x] 사용자 커스텀 파라미터 입력 가능

---

## US-004: 백테스팅 시작

**As a** 주식투자자  
**I want to** 설정한 종목과 전략으로 즉시 백테스팅을 시작할 수 있도록  
**So that** 투자 전략이 실제로 수익이 나는지 빠르게 검증할 수 있다

**수락 조건**:
- [x] 웹브라우저에서 "백테스팅 시작" 또는 "시뮬레이션 실행" 버튼 표시
- [x] 버튼 클릭 시 백엔드 시뮬레이션 엔진 호출
- [x] 진행 중 상태 표시 (예: 처리 중인 종목, 진행률)

---

## US-005: 실시간 진행상황 모니터링

**As a** 주식투자자  
**I want to** 백테스팅이 실시간으로 어디까지 진행되었는지 볼 수 있도록  
**So that** 시스템이 정상 작동하는지 확인하고 예상 완료 시간을 알 수 있다

**수락 조건**:
- [x] 웹브라우저에서 라이브 진행상황 표시 (예: "50/200 종목 완료")
- [x] 매초 또는 수초 단위로 UI 업데이트
- [x] 현재 처리 중인 종목 명 표시
- [x] 백테스팅 중단 버튼 제공
- [x] 네트워크 연결 끊김 시 재연결 자동 시도

---

## US-006: 백테스팅 결과보고서 (요약)

**As a** 주식투자자  
**I want to** 전체 백테스팅 결과를 한눈에 요약해서 볼 수 있도록  
**So that** 투자 전략의 수익성을 빠르게 파악할 수 있다

**결과 지표**:
- 최종 수익/손해 (원)
- 수익률 (%)
- 투자금 1천만 원 대비 순닷값
- 거래 횟수 (수익/손해 구분)
- 누적 거래 수익액
- 누적 거래 손해액
- 거래일
- 비용 포함 정보 (세금 0.2%, 수수료 0.011%)

**수락 조건**:
- [x] 백테스팅 완료 시 결과 화면(dashboard) 자동 표시
- [x] 모든 지표 숫자로 명확히 표시
- [x] 표, 차트 형식으로 시각화 옵션
- [x] 결과 내보내기(CSV/PDF)는 차후 추가 예정

---

## US-007: 특정 종목 선택 및 백테스팅

**As a** 주식투자자  
**I want to** 특정 종목(예: 삼성전자, SK하이닉스)에만 투자 전략을 백테스팅할 수 있도록  
**So that** 내가 집중하려는 종목에 대해 더 정밀한 검증을 할 수 있다

**배경**:
- 기존의 "코스피 200 일괄 백테스팅" 또는 "블랙/화이트 리스트 백테스팅" 외에
- 사용자가 직접 선택한 특정 종목들만 백테스팅 가능하도록 지원

**수락 조건**:
- [ ] 웹브라우저에서 "특정 종목 선택" 탭/섹션 제공
- [ ] 종목 추가 버튼: 다중 선택(Ctrl+클릭 또는 체크박스) 지원
- [ ] 선택된 종목 목록 표시 (현재 선택수 표시, 예: "2개 선택: 삼성전자, SK하이닉스")
- [ ] 종목 제거 버튼 (개별 제거 또는 전체 초기화)
- [ ] "특정 종목 모드"로 전환 시, 기존 "전체 코스피 200" 또는 "블랙/화이트 리스트" 모드와의 전환 UI 명확화
  - 예: 라디오 버튼 "전체 종목 | 블랙/화이트 기반 | **특정 종목만**"
- [ ] 특정 종목만 선택 후 전략 선택 → "백테스팅 시작" 시 해당 종목들에만 백테스팅 수행
- [ ] 선택된 종목은 서버에 저장되어 페이지 새로고침 후에도 유지
- [ ] 특정 종목 백테스팅 완료 후 결과보고서에서 선택된 종목만 표시
